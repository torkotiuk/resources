Authentication/authorization:

  • bcrypt lib working with stable nodeJS version
  • bcryptjs lib working with current (unstable) nodeJS version
    • Token consist:
    1. headers - algorithm and way of token creation (by default using const headers = {"alg": "HS256", "typ": "JWT"})
    2. payload - part that is encrypted ("id" f.e.)
    3. SECRET_KEY - usual string with any text

      4. f.e const token = 'qwertyuiop.asdfghjkl.zxcvbnm', where "qwertyuiop" - headers encrypted, "asdfghjkl" - payload encrypted, "zxcvbnm" - SECRET_KEY encrypted

      method from jsonwebtoken decrypts token and pull decrypted payload

Lesson 6 - Flexbox
  • - When we set height for block element, we would NOT use align-items in the same block element
  • - if we f.e. have font-size in document 16px that we will have additional space in inline elements on right side , in inline element <img> this additional space at bottom, by 1/4, so it's 4px. That's why in many projects make for tag <img> display: block like: 
              
            img {
             display: block;
            }
  • - Learn how to take out last margin in flex elements 
                
    
            /* Approach 1: */
            .nav-site .item {
              margin-right: 40px;
            }
            .nav-site .item:last-child {
              margin-right: 0;
            }

                      /* Approach 2: */
                      .nav-site .item:not(:last-child) {
                        margin-right: 40px;
                      }

            /* Approach 3: */
            /* use for all items that have before them element with the same class .item,  
            in other words for all element in this container except the first*/
            .nav-site .item + .item {
              background-color: rgb(227, 255, 226);
              margin-left: 40px;
            }

    How to use a parcel template

    1. Create own repository on github and clone to your pc
    2. Copy all files from original parcel-project-template https://github.com/parcel-project-template
    3. Add next scripts to package.json:
      • "predeploy": "npm run build -- --public-url /resources/",
      • "deploy": "gh-pages -d dist"
    4. change in package.json next lines
      • change username and name of project to yours --- "homepage": "https://YOUR-USERNAMA.github.io/NAME-REPOSITORY"
      • also change name of the project in "predeploy" --- "predeploy": "npm run build -- --public-url /NAME-REPOSITORY/"
    5. Install gh-pages: npm i gh-pages -D
    6. Install all dependencies: npm i

  6. Type "repo-name" & choose a necessary one
  7. Choose branch and press "Enable Automatically Deploys", then press next down button "Deploy Branch"
  8. After finishing deploy press "View" or "Open App" to see your page
