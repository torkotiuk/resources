GitHub settings on PC

If we use terminal or GitHub desctop, setting user.email means from what github account will be sent the commit

  • Check all settings:

    • - git config --list

  • Look your git user name and your user email:

    • - git config user.email
    • - git config user.name

  • Change user.email:

    • - git config --global user.email "YOUR-EMAIL@gmail.com"
