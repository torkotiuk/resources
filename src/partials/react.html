Linting Settings for React

  1. goitacademy/react-lint-config
  2. install 3 packages npm install --save-dev prettier husky lint-staged
  3. create 3 files inside your React project: .huscyrs, .lintstagedrc, .prettierrc
  4. check if installed plugin "Formatting Toggle" in VSCode
  5. Set VSCode according point 1.: Settings>codeActionsOnSave>Edit in settings.json>"editor.codeActionsOnSave": (was 'null', change to {"source.fixAll.eslint": true})
