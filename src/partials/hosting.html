Heroku: (mostly backend hosting)

  1. Create separate rep. on github for this backend
  2. Create new project on heroku heroku
  3. Choose "App name" & "Choose a region" and push "Create app" button
  4. On automatically chosen tab "Deploy" push the icon (button) GitHub
  5. If you at the first time you should push "Connect" button for binding github account
  6. Type "repo-name" & choose a necessary one
  7. Choose branch and press "Enable Automatically Deploys", then press next down button "Deploy Branch"
  8. After finishing deploy press "View" or "Open App" to see your page
